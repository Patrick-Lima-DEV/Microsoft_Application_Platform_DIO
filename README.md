# ☁️ Projeto – Microsoft Application Platform (Azure)

## 📌 Objetivo
Projeto desenvolvido durante o curso da DIO com foco em compreender os conceitos de:
- IaaS
- PaaS
- SaaS
- Azure App Service
- Deploy de aplicações na nuvem
- Arquitetura em nuvem

## 🏗 Conceitos Aplicados
### 🔹 IaaS (Infraestrutura como Serviço)
Exemplo: Azure Virtual Machines

### 🔹 PaaS (Plataforma como Serviço)
Exemplo: Azure App Service

### 🔹 SaaS (Software como Serviço)
Aplicações prontas consumidas via navegador

## 🚀 Implementação
- Criação do recurso no Azure
- Configuração do App Service
- Deploy da aplicação
- Testes de funcionamento
- Ajustes de configuração

## 📷 Prints do Projeto
Inclua imagens do portal Azure, App Service criado, aplicação rodando, etc.
- Tela de criação do App Service
- Aplicação publicada
- Configuração do ambiente

## 💡 Insights e Aprendizados
- Diferença prática entre IaaS e PaaS
- Como o Azure gerencia infraestrutura no modelo PaaS
- Facilidade de deploy comparado a uma VM tradicional
- Escalabilidade automática
- Importância da separação entre código e infraestrutura

## 🔮 Possíveis Melhorias
- Implementar CI/CD com GitHub Actions
- Adicionar banco de dados Azure SQL
- Configurar Application Insights
- Implementar autenticação via Azure AD
- Automatizar infraestrutura com ARM ou Bicep
