# Melhorias e Aprendizados

## Melhorias Possíveis
- Implementar CI/CD com GitHub Actions
- Adicionar banco de dados Azure SQL
- Configurar Application Insights para monitoramento
- Implementar autenticação via Azure AD
- Automatizar infraestrutura com ARM ou Bicep

## Insights e Aprendizados
- Diferença prática entre IaaS e PaaS
- Facilidade de deploy no Azure App Service
- Escalabilidade automática
- Separação entre código e infraestrutura
- Documentação clara facilita manutenção e evolução
